## 线程通信：生产者与消费者问题

### 解决方法

#### 管程法（缓冲区）

##### 思想：

利用wait()、notify()、notifyAll()方法完成；

当生产者判断当前产品数量小于最大容量时，生产产品并通知消费者消费；否则等待消费者消费；

当消费者判断当前产品数量大于0时，消费产品并通知生产者生产产品；否则等待生产者生产；

##### 代码：

```java
/**
 * @program: JUC
 * @description: 解决方法1 管程法（缓冲区）
 * @author: Mr.Wang
 * @create: 2022-01-15 14:31
 **/
public class Test1 {

    public static void main(String[] args) {

        SynContainer synContainer = new SynContainer();

        new Productor(synContainer).start();
        new Consumer(synContainer).start();
    }

}

//生产者
class Productor extends Thread{

    SynContainer synContainer;

    public Productor(SynContainer synContainer){
        this.synContainer=synContainer;
    }

    @Override
    public void run() {
        for (int i = 1; i < 100; i++) {
            System.out.println("生产了"+i+"只鸡");
            synContainer.produce(new Product());
        }
    }
}

//消费者
class Consumer extends Thread{
    SynContainer synContainer;

    public Consumer(SynContainer synContainer){
        this.synContainer=synContainer;
    }

    @Override
    public void run() {
        for (int i = 1; i < 100; i++) {
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
            System.out.println("消费"+i+"只鸡");
            synContainer.consumer();
        }
    }
}

//产品
class Product extends Thread{
}

//缓冲台
class SynContainer {

    Product[] product = new Product[10];
    int count = 1;

    //    消费
    public synchronized void consumer() {
        System.out.println("生产容量池中"+count);
        if (count > 0) {
            //消费
            count--;
            notifyAll();
        } else {
            //等待
            try {
                this.wait();
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
    }

    //    生产
    public synchronized void produce(Product products) {
        System.out.println("消费容量池中"+count);
        if (count < this.product.length) {
            //生产
            product[count]=products;
            count++;
            notifyAll();
        } else {
            //等待
            try {
                this.wait();
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
    }
}

```



#### 信号灯法

##### 思想：

通过一个标志位，来实现双方的通信；

比如对于生产者，判断标志位如果是TRUE，则生产产品并通知消费者消费；否则等待消费者消费；

对于消费者，判断标志位如果是TRUE，则等待生产者生产；否则消费产品并通知生产者生产；

##### 代码：

```java
/**
 * @program: JUC
 * @description: 信号灯法 利用标志位解决
 * @author: Mr.Wang
 * @create: 2022-01-15 15:14
 **/
public class Test2 {

    public static void main(String[] args) {
        TV tv = new TV();
//        ExecutorService service = Executors.newFixedThreadPool(3);
//        service.submit(new player(tv));
//        service.submit(new watcher(tv));
//        service.shutdown();
        new player(tv).start();
        new watcher(tv).start();
    }
}

//生产者 演员
class player extends Thread {
    TV tv;

    public player(TV tv) {
        this.tv = tv;
    }

    @Override
    public void run() {
        for (int i = 0; i < 20; i++) {
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
            if (i % 2 == 0) {
                tv.play("天天向上");
            } else {
                tv.play("快乐大本营");
            }
        }
    }
}

//消费者 观众
class watcher extends Thread {
    TV tv;

    public watcher(TV tv) {
        this.tv = tv;
    }

    @Override
    public void run() {
        for (int i = 0; i < 20; i++) {
            tv.watch();
        }
    }
}

//产品 节目
class TV {
    //演员表演，观众等待
    //观众观看，演员表演
    String programmer;
    //默认演员表演
    boolean flag = true;

    public synchronized void play(String programmer) {
        if (!flag) {
            //此时观众在观看
            try {
                this.wait();
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        } else {
            //演员表演
            System.out.println("正在表演的是" + programmer);
            //通知观众观看
            this.programmer = programmer;
            this.flag = !flag;
            this.notifyAll();
        }
    }

    public synchronized void watch() {
        if (flag) {
            //演员在表演
            try {
                this.wait();
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        } else {
            //观众观看
            System.out.println("正在观看的是：" + this.programmer);
            this.flag = !flag;
            this.notifyAll();
        }
    }

}
```
